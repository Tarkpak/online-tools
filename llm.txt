# 在线工具集项目总结

## 项目概述
一个纯前端实现的在线工具集合网站，无需后端服务，可直接在浏览器中运行。

## 核心功能

### 1. 图片转线稿工具
- 使用 OpenCV.js 进行图像处理
- 支持实时预览和参数调节
- 提供原图与线稿对比功能

### 2. 国际化支持
- 支持中文、英文、日语三种语言
- 使用 i18next 实现多语言切换
- 自动检测浏览器语言

## 技术栈

### 前端框架与库
- HTML5/CSS3/JavaScript (原生实现)
- OpenCV.js (图像处理)
- i18next (国际化)
- AOS (滚动动画)

### 样式特点
- 响应式设计
- 现代化动画效果
- 渐变背景
- 毛玻璃效果

## 关键代码片段

### 1. 多语言切换
```javascript
i18next.init({
    fallbackLng: 'zh',
    resources: {
        zh: { translation: {...} },
        en: { translation: {...} },
        ja: { translation: {...} }
    }
});
```

### 2. 图片处理
```javascript
let src = cv.imread(canvas);
let dst = new cv.Mat();
cv.cvtColor(src, src, cv.COLOR_RGBA2GRAY);
cv.Canny(src, dst, 50, 150, 3, false);
cv.bitwise_not(dst, dst);
```
